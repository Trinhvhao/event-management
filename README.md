# event-management

Hệ thống quản lý sự kiện đại học với tính năng check-in QR code, quản lý điểm rèn luyện và cấp chứng nhận tự động.

## Công nghệ sử dụng

### Backend
- Node.js + Express + TypeScript
- PostgreSQL + Prisma ORM
- JWT Authentication

### Frontend
- Next.js 14 + TypeScript
- TailwindCSS
- Framer Motion

## Cài đặt

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Tính năng chính

- ✅ Check-in QR Code thần tốc
- ✅ Dashboard real-time
- ✅ Quản lý điểm rèn luyện
- ✅ Cấp chứng nhận tự động
- ✅ Bảo mật đa cấp độ
